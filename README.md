# Aplicación de calculadora de IOS

## Reseña
Este es un desarrollo impulsado por "Fernando Herrera" en el curso ["React Native: Aplicaciones nativas para IOS y Android"](https://www.udemy.com/course/react-native-fh/)
También se pueden encontrar este y otros cursos en:
- [Web](https://fernando-herrera.com/)
- [Youtube](https://www.youtube.com/@DevTalles)

## Tecnologia
Proyecto desarrollado con:
- Node js: 20.9.0
- TypeScript: 4.8.4
- React: 18.2.0
- React Native: 0.72.7
- Metro: 0.76.8

## Scripts
Para despachar cualquiera de los siguientes script anteponer `npm run`, por ejemplo `npm run start`
- android1: para ejecutar el emulador android preferido. Modifique Pixel_7_API_33 por el nombre del suyo
- android2: para despachar el servidor de react-native en Android y se conecte al dispositivo o emulador anterior
- ios: para despachar el servidor de react-native en iOS y se conecte al dispositivo o emulador seleccionado. Modifique 'iPhone 14' por el suyo
- lint: para buscar errores de formato y/o sintaxis con eslint
- start: despachar servidor Metro
